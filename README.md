# 🌍 Fog of War Map (Персональная карта открытий)

> 🤖 **AI-Driven Project:** Весь исходный код этого проекта (100%) был сгенерирован с помощью ИИ. 
Моя роль заключалась в проектировании архитектуры, системном администрировании (Docker, Nginx, Cloudflare, PostGIS), дебаггинге и управлении ИИ-ассистентом.

Интерактивная карта мира, покрытая "туманом войны". Туман рассеивается только в тех местах, где вы физически побывали. Приложение позволяет загружать реальные GPS-треки (GPX) или закрашивать исследованные зоны вручную, прямо как кистью в Paint.

## ✨ Основные возможности
* **🌫️ Настоящий туман войны:** Весь мир покрыт темной полупрозрачной пеленой, которая открывается по мере ваших путешествий.
* **🗺️ Загрузка GPX-треков:** Загружайте маршруты с часов, телефона или навигатора. Трек автоматически превращается в "исследованный коридор" шириной 30 метров (радиус 15м).
* **🖌️ Режим свободной кисти (Paint):** Можно закрашивать пройденные участки вручную, просто ведя мышкой по карте.
* **⏹️ Точные полигоны:** Инструменты для выделения целых парков, районов или кварталов (Leaflet-Geoman).
* **↩️ Система отмены (Undo):** Случайно закрасили лишнее? Одно нажатие, и последнее действие отменяется с полным восстановлением тумана.

## 🛠️ Технологический стек
**Backend:**
* [FastAPI](https://fastapi.tiangolo.com/) — сверхбыстрый современный веб-фреймворк.
* [PostgreSQL](https://www.postgresql.org/) + [PostGIS](https://postgis.net/) — база данных с мощнейшим пространственным движком для высчитывания пересечений полигонов и тумана.
* [GeoAlchemy 2](https://geoalchemy-2.readthedocs.io/) & SQLAlchemy — ORM и работа с геометрией.
* JWT Auth (bcrypt) — безопасная авторизация.

**Frontend:**
* Vanilla JavaScript, HTML5, CSS3 — легковесно и быстро, без сложных сборщиков.
*[Leaflet](https://leafletjs.com/) — картографический движок (подложка 2GIS / CartoDB).
* [Leaflet-Geoman](https://geoman.io/) — инструменты для рисования на карте.

**DevOps:**
* Docker & Docker Compose — полная изоляция приложения.
* Nginx — реверс-прокси и раздача статики.

## 🚀 Установка и запуск (Docker)

Проект полностью контейнеризован. Для запуска вам понадобится только установленный `docker` и `docker-compose`.

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/ВАШ_НИК/fog-of-war-map.git
   cd fog-of-war-map
   ```
2. **Настройте переменные окружения:**
   В папке backend создайте файл .env (или отредактируйте существующий) и задайте надежные пароли:
   ```DATABASE_URL=postgresql://foguser:СЕКРЕТНЫЙ_ПАРОЛЬ_БД@db:5432/fogmap
   SECRET_KEY=СЛУЧАЙНЫЙ_НАБОР_СИМВОЛОВ_ДЛЯ_JWT
   ALGORITHM=HS256
   ACCESS_TOKEN_EXPIRE_MINUTES=1440
   ```
   Не забудьте также обновить пароль к БД в файле docker-compose.yml (POSTGRES_PASSWORD).
3. **Запустите проект:**
   ```bash
   docker compose up -d --build
   ```
## Как сбросить забытый пароль?
Шаг 1. Запускаем интерактивный Python прямо внутри контейнера бэкенда:
```bash
docker compose exec -it backend python
```
(У тебя откроется консоль Python, строчки будут начинаться с >>>)

Шаг 2. Вводим команды по очереди (поменяй email и новый пароль на свои):
```python
from app.database import SessionLocal
from app.models import User
from app.auth import get_password_hash

# Подключаемся к базе
db = SessionLocal()
# Находим пользователя по email
user = db.query(User).filter(User.email == "твой@email.ru").first()
# Меняем пароль (сразу шифруя его)
user.hashed_password = get_password_hash("НовыйПароль123")
# Сохраняем изменения
db.commit()
# Выходим
exit()
```
